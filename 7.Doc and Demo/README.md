## 📌 Project Overview

This project analyzes **electric vehicle market trends** and presents interactive insights through **Tableau dashboards** integrated into a **Flask web application**.

The project helps users understand:

| Insight | Description |
|---|---|
| 🏎️ **EV Brand Performance** | Compare brands across key performance metrics |
| ⚙️ **Vehicle Efficiency** | Analyze efficiency (Wh/km) across models |
| 🔋 **Driving Range** | Explore top-performing EVs by range |
| ⚡ **Fast Charging Capability** | Identify fastest-charging EV brands |
| 💰 **Price Distribution** | Understand pricing trends across the market |
| 🚙 **Body Style Distribution** | See which body styles dominate the EV market |

## 📖 Story Insights

> Key takeaways uncovered through Tableau's story-driven analysis:

- 🏎️ **Tesla** offers high performance across multiple metrics.
- ⚡ **Porsche** provides excellent charging speed.
- 🔋 **Lucid** offers the highest driving range.
- 🚙 **SUV** body style dominates the EV market.
- 💰 EV prices **vary significantly** across manufacturers.
